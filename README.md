# AlxStuff

**Kill boredom in style.** A collection of interactive web tools and games to waste time productively.

## What's Inside

- **Cat Meme Generator** - Random cat memes with customizable text overlays
- **Joke Machine** - Jokes across multiple categories (programming, puns, spooky, Christmas)
- **Would You Rather** - 50 thought-provoking dilemmas
- **Useless Facts** - Random trivia that'll blow your mind
- **Mini Games** 
  - Reaction Time Test
  - Color Matcher (guess hex codes)
  - Number Memory Challenge
- **Random Challenges** - Daily tasks with streak tracking
- **This or That** - Quick preference polls
- **Boredom Meter** - Rate your boredom, get activity suggestions

```bash

# Install dependencies
npm install

# Start dev server
npm run dev
```
